# ExamSite.in - AI-Powered Government Job Portal

ExamSite.in is a comprehensive platform for government job seekers in India, providing real-time updates on Sarkari Naukri, Admit Cards, Results, and Syllabi. The platform features an intelligent AI assistant powered by Google Gemini to help users with their queries.

## Features

*   **Latest Job Updates:** Real-time notifications for SSC, UPSC, Railway, Banking, Police, and Defence jobs.
*   **AI Assistant:** Built with Google Gemini API to answer user queries, summarize notifications, and provide voice guidance.
*   **Bilingual Support:** Full support for English and Hindi languages.
*   **Smart Search:** Efficient search functionality to find specific exams or notifications.
*   **Categorized Information:** Organized sections for Admit Cards, Results, Answer Keys, and Admissions.
*   **Accessibility:** Text-to-Speech (TTS) features for reading out notifications.

## Tech Stack

*   **Frontend:** React (TypeScript)
*   **Styling:** Tailwind CSS
*   **Icons:** Lucide React
*   **AI Integration:** Google GenAI SDK (`@google/genai`)

## Setup Instructions

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/contactsarkarikhabar-sys/examsite.git
    ```

2.  **Install Dependencies:**
    ```bash
    npm install
    ```

3.  **Environment Variables:**
    The app uses `process.env.API_KEY` for Google GenAI features.

4.  **Run the Project:**
    ```bash
    npm start
    ```

This is a demo application created for educational purposes. Please verify all job-related information from official government websites.
